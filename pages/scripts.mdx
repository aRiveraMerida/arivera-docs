# Scripting con NodeJS

<Divider />


Un script es un conjunto de instrucciones o comandos que se utilizan para automatizar tareas en un programa o sistema. Estas instrucciones están escritas en un lenguaje de programación y se ejecutan en un orden específico para lograr un objetivo determinado. Los scripts se utilizan comúnmente para realizar tareas repetitivas o para simplificar procesos complejos, como la gestión de archivos o la configuración de sistemas. También se utilizan en el desarrollo de sitios web para mejorar la funcionalidad y la interactividad.
